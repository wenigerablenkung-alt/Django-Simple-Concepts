📖 How Sessions Work
# In views.py:

# WRITE to session
request.session['cart'] = ['item1', 'item2']
request.session['user_pref'] = 'dark'

# READ from session
cart = request.session.get('cart', [])

# DELETE from session
del request.session['cart']

# CLEAR everything
request.session.flush()

# Sessions are stored in the DATABASE
# Browser gets a "sessionid" COOKIE
# Django looks up that ID in the DB
